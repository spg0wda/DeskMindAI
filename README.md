# AgentK - Enterprise Multi-Agent Service Desk Chatbot

AgentK is an AI-powered enterprise service desk chatbot built using LangGraph Supervisor Architecture.

## Project Idea

The system uses multiple specialized AI agents to classify user issues, gather requirements, generate solutions, collect feedback, and improve future responses using a feedback loop.

## Main Features

- Multi-agent architecture
- LangGraph supervisor workflow
- Domain classification
- Requirement gathering
- Ticket priority detection
- Feedback-based learning
- Domain-based prompt memory
- Dashboard for analytics

## Tech Stack

- Backend: FastAPI
- Frontend: React + Vite
- Database: MySQL
- AI Workflow: LangGraph
- LLM: Groq / Gemini
- Version Control: Git + GitHub
